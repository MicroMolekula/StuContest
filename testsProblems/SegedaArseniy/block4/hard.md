Задача 3 (сложный):
Набор [1, 2, 3, ..., n] содержит в общей сложности n! уникальные перестановки.
Перечислив и пометив все перестановки по порядку, мы получим следующую последовательность для n = 3:
"123"
"132"
"213"
"231"
"312"
"321"
Учитывая n и k, вернуть последовательность перестановок. kth
 
Пример 1:
Вход: n = 3, k = 3
Выход: «213»
Пример 2:
Вход: n = 4, k = 9
Выход: «2314»
Пример 3:
Вход: n = 3, k = 1
Выход: «123»
 
Ограничения:
1.	1 <= n <= 9
2.	1 <= k <= n!
