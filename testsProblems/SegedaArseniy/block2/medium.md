Задача 2 (средний):
Реализуйте pow(x, n), который вычисляет x в степени n (т.е. x^n).
 
Пример 1:
Ввод: x = 2,00000, n = 10
Выход: 1024,00000
Пример 2:
Вход: x = 2,10000, n = 3
Выход: 9,26100
Пример 3:
Ввод: x = 2,00000, n = -2
Выход: 0,25000
Объяснение: 2 -2 = 1/2 2 = 1/4 = 0,25
 
Ограничения:
1.	-100.0 < x < 100.0
2.	-2^31 <= n <= (2^31)-1
3.	-10^4 <= xn <= 10^4
